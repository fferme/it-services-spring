# Spring Boot CRUD Project
This project is a CRUD (Create, Read, Update, Delete) application built with Spring Boot. It allows users to manage orders, services, and clients.

# Features
- Add, view, update, and delete orders
- Add, view, update, and delete service items
- Add, view, update, and delete clients

- Managing Clients
